Batalla Naval

Bienvenido a Batalla Naval

Proyecto web desarrollado con Spring Boot, Thymeleaf, JPA, MySQL, Spring Security y WebSocket.

Regístrate, inicia sesión, crea una partida y enfréntate a otro jugador en tiempo real.

Iniciar sesión Registrarse Ver partidas

Registro y login

Los usuarios podrán crear cuenta e iniciar sesión de forma segura.

Crear partidas

Cada usuario autenticado podrá crear una nueva partida con código.

Juego por turnos

Los jugadores colocarán sus barcos y dispararán por turnos.

Tiempo real

Los disparos y cambios de turno se verán al momento con WebSocket.
