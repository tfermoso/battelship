Bienvenido a Batalla Naval

Proyecto web desarrollado con Spring Boot, Thymeleaf, JPA, MySQL, Spring Security y WebSocket.

Regístrate, inicia sesión, crea una partida y enfréntate a otro jugador en tiempo real en un tablero naval con turnos sincronizados.

Radar táctico

Prepara tu flota, busca al enemigo y coordina tus disparos para dominar el océano.

🔐

Registro y login

Los usuarios podrán crear cuenta e iniciar sesión de forma segura para acceder al juego.

⚓

Crear partidas

Cada usuario autenticado podrá crear una nueva partida y compartir su código con otro jugador.

🎯

Juego por turnos

Los jugadores colocarán sus barcos, atacarán por turnos y buscarán hundir la flota rival.

📡

Tiempo real

Los disparos, cambios de turno y eventos principales se sincronizan al momento con WebSocket.
